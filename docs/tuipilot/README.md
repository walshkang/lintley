Docs for the TUI assistant (tuipilot).

This folder follows the acerca-trip-planner conventions and contains the canonical docs used by contributors:

- context.md — repo context, constraints, and privacy guidance
- agents.md — background agents, responsibilities, and interfaces
- design.md — UI design, panes, keybindings, and accessibility notes
- visual-signals.md — visual status tokens, colors, and micro-animations

Use these as the primary guide for implementing features and slices.
